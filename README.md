# 💪 Fast Fitness - Android Workout Tracker

Fast Fitness is an Android application developed to help users track workouts, monitor fitness progress, and maintain exercise history. The app provides an intuitive interface for managing workout routines and improving fitness performance.

---

## ✨ Features

- 🏋️ Workout Tracking
- ⏱️ Stopwatch & Countdown Timer
- 📊 Exercise History
- 🎵 Music Playback Support
- 📱 Responsive Android UI
- 💾 Local Data Storage using SQLite

---

## 🛠 Tech Stack

- Java
- Android Studio
- XML
- SQLite
- Gradle
- Git

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/Ugesh123/fast-fitness-android-app.git
```

### Open in Android Studio

Open the project and allow Gradle to sync.

### Run

Connect an Android device or emulator and click **Run**.
